# 用户接口

## 当前用户资料

`GET /api/user/profile`

### 请求头

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| Authorization | string | 是 |  | `Bearer <token>` |

### 安全字段

| 项目 | 字段 |
|------|------|
| 请求签名字段 | 不参与签名 |
| 请求加密字段 | 不参与加密 |
| 响应签名字段 | 不参与签名 |
| 响应加密字段 | email, phone |

### 响应示例

```json
{
  "status": true,
  "code": 1110,
  "message": "获取成功",
  "data": {
    "id": 1,
    "username": "demo_user",
    "nickname": "Demo",
    "email": "demo@example.com",
    "phone": "",
    "avatar": "",
    "status": 1,
    "lastLoginAt": "2026-06-02 12:00:00",
    "lastLoginIp": "127.0.0.1",
    "createdAt": "2026-06-02 12:00:00",
    "updatedAt": "2026-06-02 12:00:00"
  }
}
```

### data 内部字段

| 字段 | 类型 | 是否必须 | 默认值 | 字段解释 |
|------|------|----------|--------|----------|
| id | int | 是 |  | 用户 ID |
| username | string | 是 |  | 用户名 |
| nickname | string | 是 |  | 昵称 |
| email | string | 是 |  | 邮箱 |
| phone | string | 是 |  | 手机号 |
| avatar | string | 是 |  | 头像 |
| status | int | 是 |  | 状态：1 正常，0 禁用 |
| lastLoginAt | string | 否 |  | 最后登录时间 |
| lastLoginIp | string | 否 |  | 最后登录 IP |
| createdAt | string | 是 |  | 创建时间 |
| updatedAt | string | 是 |  | 更新时间 |
